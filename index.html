AgentVMThe runtime your AI agents deserve

Process management, memory, tools, messaging, and scheduling for autonomous AI agents. Framework-agnostic. Zero lock-in.

Get Started
View on GitHub
🔄

Process Manager

Spawn, pause, resume, and terminate agent processes. Each agent runs with its own lifecycle, memory, and crash recovery.

🧠

Pluggable Memory

In-memory or SQLite persistence out of the box. Agents remember across restarts. Plug in your own backend.

🔧

Tool Router

Register tools with rate limiting, permissions, and side-effect tracking. Agents only access what you allow.

🤖

LLM Agents

createLLMAgent() wires up Anthropic or OpenAI with an automatic tool-use loop. Multi-agent pipelines in 10 lines.

🔌

MCP Integration

Connect to any MCP server (stdio or SSE). Tools auto-register with the kernel. Claude Desktop, Cursor, and more.

🔗

Framework Adapters

Use AgentVM tools in LangChain, Vercel AI SDK, OpenAI, or Anthropic. Zero-dependency adapters that produce plain objects.
